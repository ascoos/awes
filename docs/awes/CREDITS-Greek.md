# Συντελεστές
Το αρχείο αυτό αποτελεί ένα συνοδευτικό αρχείο που περιέχει πληροφορίες για τους συντελεστές δημιουργίας και μετάφρασης, τους πόρους και τις βιβλιοθήκες που χρειάστηκαν για την ολοκλήρωση του AWES.

Κάθε νέο πρόσωπο ή στοιχείο που συμμετέχει στην διαδικασία δημιουργίας ή αναβάθμισης του προγράμματος αναφέρεται, ως οφείλουμε, στο αρχείο αυτό. 

## Ομάδα Ανάπτυξης
- `Χρήστος Δρογκίδης` (Προγραμματιστής) - Ο δημιουργός του AWES

## Συγγραφείς
- Όλα τα αρχικά κείμενα του AWES (Ελληνικά και Αγγλικά) έχουν συνταχθεί από τον δημιουργό του AWES.

## Μεταφραστές
- Οι αρχικές γλώσσες (Ελληνικά και Αγγλικά) έχουν συνταχθεί από τον δημιουργό του AWES.

## Βιβλιοθήκες και Πόροι
- `Original Apache Web Server`  [ https://www.apache.org ]   HTTP Διακομιστής ιστού.
- `Apache Lounge`  [ https://www.apachelounge.com ]   Το Apache Lounge παρέχει ενημερωμένα δυαδικά αρχεία του Apache για Windows.
- `NGINX`  [ https://nginx.org ]   HTTP Διακομιστής ιστού.
- `PHP`  [ https://www.php.net ]   Μια δημοφιλής γλώσσα γενικής χρήσης, ιδιαίτερα κατάλληλη για την ανάπτυξη ιστοσελίδων.
- `Perl`   [ https://www.perl.org ]   Η Perl είναι μια εξαιρετικά ικανή, πλούσια σε χαρακτηριστικά γλώσσα προγραμματισμού.
- `Python`  [ https://www.python.org ]   Η Python είναι μια γλώσσα προγραμματισμού για γρήγορη και αποτελεσματική εργασία.
- `NodeJS`  [ https://nodejs.org ]   Εκτελέστε JavaScript παντού
- `MariaDB`  [ https://mariadb.org ]   Η καινοτόμος βάση δεδομένων ανοιχτού κώδικα
- `PostgreSQL`  [ https://www.postgresql.org ]   Η πιο προηγμένη σχεσιακή βάση δεδομένων ανοιχτού κώδικα στον κόσμο.
- `MongoDB`  [ https://www.mongodb.com ]   Αγαπήθηκε από τους προγραμματιστές. Σχεδιασμένο για σύγχρονες εφαρμογές.
- `FileZilla`  [ https://filezilla-project.org ]   Δωρεάν ανοιχτού κώδικα FTP και FTPS διακομιστής.
- `hMailServer`  [ https://www.hmailserver.com ]   Δωρεάν, ανοιχτού κώδικα, διακομιστής ηλεκτρονικού ταχυδρομείου για Microsoft Windows.
- `SubVersion`  [ https://subversion.apache.org ]   Δωρεάν, ανοιχτού κώδικα, σύστημα ελέγχου εκδόσεων.
- `phpMyAdmin`  [ https://www.phpmyadmin.net ]   Προορίζεται να χειριστεί τη διαχείριση του MariaDB, μέσω του Ιστού.
- `Memcached`  [ https://www.memcached.org ]   Δωρεάν, ανοιχτού κώδικα, σύστημα προσωρινής αποθήκευσης αντικειμένων.
- `Browscap` [ https://browscap.org ]   Πληροφοριακή βιβλιοθήκη δυνατοτήτων προγράμματος περιήγησης.
- `Tiny File Manager`  [ https://tinyfilemanager.github.io ]   Διαχειριστείτε τα αρχεία σας αποτελεσματικά και εύκολα.
- `WebSVN`  [ https://websvnphp.github.io ]   Προσφέρει μια προβολή στα αποθετήρια Subversion.
- `SVGRepo`  [ https://www.svgrepo.com ]   Ανοικτής άδειας SVG Vector και εικονίδια.
- `Notepad++`  [ https://notepad-plus-plus.org ]   Δωρεάν πρόγραμμα επεξεργασίας πηγαίου κώδικα.
- `Visual Studio Code`  [ https://code.visualstudio.com ]   Το VSCode υποστηρίζει σχεδόν κάθε μεγάλη γλώσσα προγραμματισμού.
- `Firefox Developer Edition`  [ https://www.mozilla.org/en-US/firefox/developer ]   Το πρόγραμμα περιήγησης που έχει φτιαχτεί για προγραμματιστές
- `SnipTool`  [ https://www.reasyze.com/sniptool ]   Καταγράψτε περιοχές της επιφάνειας εργασίας σας και σχολιάστε
- `XDebug`  [ https://xdebug.org ]   Το Xdebug είναι μια επέκταση για την PHP και παρέχει μια σειρά χαρακτηριστικών για τη βελτίωση της εμπειρίας ανάπτυξης της PHP.
