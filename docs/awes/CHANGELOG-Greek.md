# 24.1.1 [1409]

!!DATE 2024-10-31

## AWES
- [ + ] Προστέθηκε η αυτόματη εγκατάσταση των απαραίτητων πιστοποιητικών SSL στο Root Store των windows,
        ώστε να μη "κρεμάνε" οι σελίδες του AWES που τρέχουν στο localhost.
        Για τις σελίδες python.loc και node.loc απαιτούνται πρόσθετα πιστοποιητικά που η δημιουργία τους
        θα ενεργοποιηθεί σε επόμενες εκδόσεις.

- [ + ] Προστέθηκε επιλογή για δημιουργία προβολής των περιεχομένων του τομέα ως ευρετηρίο (Indexes).
        Η υλοποίησή του γίνεται κατά την δημιουργία του τομέα μέσα από το αντίστοιχο πλαίσιο διαλόγου.
        Αυτό κρίνεται απαραίτητο όταν σε κάποιο τομέα, θέλουμε να βλέπουμε στον browser τα αρχεία 
        που περιέχονται στον φάκελο και να επιλέγουμε εμείς ποιο θέλουμε να εκτελεστεί.

- [ + ] Προστέθηκαν δυνατότητες χειρισμού των χρηστών. 
        Με δεξί κλικ στην λίστα πάνω στον επιλεγμένο χρήστη ανοίγει ένα αναδυόμενο μενού,
        που περιέχει διάφορες λειτουργίες γι' αυτόν :   
    - Το άνοιγμα του φακέλου δεδομένων του χρήστη, είτε στην εξερεύνηση των Windows, είτε στο Visual Studio Code
    - Αντίγραφα ασφαλείας όλων των δεδομένων του και επαναφορά αυτών.
    - Εμφάνιση του φακέλου αντιφράφων ασφαλείας του χρήστη στην εξερεύνηση των Windows.

- [ + ] Προστέθηκαν δυνατότητες χειρισμού των τομέων. 
        Με δεξί κλικ στην λίστα πάνω στον επιλεγμένο τομέα ανοίγει ένα αναδυόμενο μενού, 
        που περιέχει διάφορες λειτουργίες γι αυτό. Ενδεικτικά αναφέρουμε :
    - Το άνοιγμα του φακέλου του τομέα, είτε στην εξερεύνηση των Windows, είτε στο Visual Studio Code
    - την δημιουργία πιστοποιητικού SSL (Ενεργοποίηση σε επόμενη έκδοση)
    - Την δημιουργία ή διαγραφή αποθετηρίου (Ενεργοποίηση σε επόμενη έκδοση)
    - Την εγκατάσταση διαδικτυακής εφαρμογής (Ενεργοποίηση σε επόμενη έκδοση)
    - Αντίγραφα ασφαλείας και επαναφορά αυτών.
    - Εμφάνιση του φακέλου αντιφράφων ασφαλείας του τομέα στην εξερεύνηση των Windows.

- [ + ] Ενεργοποιήθηκε το παράθυρο αναβάθμισης. 
        Πλέον, δεν χρειάζεται να κατεβάζετε τις δευτερεύουσες ενημερώσεις χειροκίνητα, 
        αν και υπάρχει πάντα η επιλογή να σας κατευθύνει το AWES στο Sourcefoge 
        για να κατεβάσετε και να εγκαταστήσετε το πακέτο αναβάθμισης μόνοι σας.        

- [ + ] Προστέθηκε η δυνατότητα ειδοποίησης στην επιφάνεια εργασίας εάν υπάρχει διαθέσιμη αναβάθμιση. 
        Η επιλογή της δυνατότητας αυτής, γίνεται μέσα από το παράθυρο διαμόρφωσης, στην καρτέλα "Αναβαθμίσεις".

- [ ! ] Διορθώθηκαν σφάλματα στην καρτέλα διαχείρισης των τομέων των χρηστών.
    - Διορθώθηκε το σφάλμα στην διαγραφή του αρχείου διαμόρφωσης του apache όταν διαγράφουμε έναν τομέα ή υποτομέα.
    - Διορθώθηκαν σφάλματα στην αυτόματη προσθήκη και ανανέωση της λίστας των DNS του αρχείου Hosts των Windows στην αντίστοιχη καρτέλα του AWES, μετά από προσθήκη ή διαγραφή τομέων.

- [ ! ] Διορθώθηκαν σφάλματα στα links προς τις ιστοσελίδες του Ascoos.

- [ ! ] Διορθώθηκαν διάφορα μικρά σφάλματα και διενέξεις του AWES και των συστατικών εφαρμογών του.

### ΔΙΑΚΟΜΙΣΤΕΣ WEB
- [ + ] `PHP` --> Προστέθηκε η έκδοση 8.4 RC3 της PHP.

- [ ^ ] `PHP` --> 8.2.25, 8.3.13 -- Αναβαθμίστηκαν οι τελευταίες εκδόσεις. 8.2 και 8.3

- [ + ] `Memcached` --> Προστέθηκε ο δημοφιλής επιταχυντής στην έκδοση 1.6.8.

- [ + ] `MongoDB` --> Προστέθηκε η βάση δεδομένων στην έκδοση 8.0.3, μαζί με το πρόγραμμα διαχείρισης compass. Η βάση δεδομένων δεν λειτουργεί στις εκδόσεις PHP 5.6 και 8.4 (Δείτε στην ενότητα `ΜΗ ΔΙΟΡΘΩΜΕΝΑ`).

- [ + ] `SubVersion` --> Προστέθηκε η λειτουργία του ως υπηρεσία στα windows, αλλά και ως συστατικό στον apache. Το πρώτο είναι προσωρινά απενεργοποιημένο διότι παρουσιάζει ακόμη ορισμένα σφάλματα. Ως συστατικό του apache λειτουργεί και η εμφάνιση ενός αποθετηρίου μπορεί να γίνει είτε καλώντας το ως https://localhost/svn/[RepositoryName], είτε πιο ασφαλές μέσω της ενσωματωμένης εφαρμογής WebSVN που υπάρχει στην καρτέλα με τα "πρόσθετα".

- [ ^ ] `Apache` --> Ενημερώθηκε ο apache στην έκδοση 2.4.62 με openssl 3.1.7. Η έκδοση αυτή λειτουργεί με VS17 αλλά τρέχει και εφαρμογές που έχουν δημιουργηθεί με VS16. Βασικά τρέχουν όλες οι εφαρμογές PHP (5.6 - 8.4), Python και NodeJS.

- [ - ] `MariaDB` --> Αφαιρέθηκαν 5 εκδόσεις (11.0.x, 11.1.x, 11.2.x, 11.3.x, 11.5.x) καθότι κρίθηκε περιττό να υπάρχουν τόσες εκδόσεις μιας βάσης δεδομένων καθώς οι δυνατότητές τους καλύπτονται απόλυτα από τις εκδόσεις 11.4.x και 11.6.x που παρέμειναν στο AWES.
