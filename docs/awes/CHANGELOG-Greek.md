## ΒΙΒΛΙΟΘΗΚΕΣ ΚΩΔΙΚΑ

### 🟢 Ascoos Franework 24 (Free Edition)

`Απαιτεί PHP >= 8.2`

Έγινε προσθήκη μιας αρχικής 24.0.6 Pre-Alpha έκδοσης του Ascoos Framework 24 που περιλαμβάνει κλάσεις διαχείρισης πινάκων, μεθόδων ανάλυσης δεδομένων των πινάκων καθώς και δημιουργία διαγραμμάτων βάσει των δεδομένων των πινάκων, επίσης κλάση διαχείρισης ημερομηνιών κ.α. 

Αυτό το πλαίσιο εργασίας, έχει προρυθμιστεί να φορτώνεται πριν από την εκτέλεση κάθε script, οπότε δεν θα χρειαστεί να την καλείτε κάθε φορά.

Ένα άλλο χρήσιμο στοιχείο είναι πως μέσω του φακέλου libs, μπορεί να φορτώσει μαζί του και άλλες βιβλιοθήκες ή frameworks, όπως π.χ. το phpbcl8, την Laravel κλπ, αρκεί να είναι μέσα σε ένα φάκελο π.χ. `afw\libs\[φάκελος βιβλιοθήκης]\autoload.php`. Αυτόματα το Ascoos Framework θα διαβάσει το autoload.php της κάθε βιβλιοθήκης μέσα στον φάκελο `libs` και θα τις φορτώσει μαζί του.

### 🟢 phpbcl8
Η βιβλιοθήκη υποστήριξης εκτέλεσης λειτουργιών νεότερων εκδόσεων PHP 8.x σε παλαιότερες εκδόσεις της σειράς 8 (π.χ. νέες functions της PHP 8.4.1 που δεν υπάρχουν στην PHP 8.2 μπορούν να εκτελεστούν μέσω της βιβλιοθήκης phpbcl8). Η βιβλιοθήκη αυτή είναι ενσωματωμένη στο `Ascoos Framework 24`, οπότε διαβάζεται αυτόματα κατά την φόρτωση του framework.
