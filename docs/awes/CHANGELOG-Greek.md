!!DATE 2024-11-20


## ΣΧΕΤΙΚΑ ΜΕ ΑΥΤΗ ΤΗΝ ΕΚΔΟΣΗ
Από την έκδοση αυτή και νεότερες, το παρών αρχείο αλλαγών θα περιλαμβάνει μόνο τις αλλαγές 
που έχουν γίνει στην έκδοση που εκτελείτε. 
Αυτό θεωρήθηκε επιβεβλημένο, ώστε να υπάρχει μεγαλύτερη ανάλυση των αλλαγών χωρίς την επιβάρυνση των πόρων του συστήματος.

Συγκεντρωτικά, όλες τις αλλαγές θα μπορείτε να τις βλέπετε σύντομα μέσω 
της ιστοσελίδας https://awes.ascoos.com/whatsnews/changes.php (Αυτή τη στιγμή είναι υπό κατασκευή).
Για την εύκολη πρόσβαση σε αυτή, προστέθηκε το μενού `Help --> All Changes`, ακριβώς κάτω από το μενού του παρόντος.

## AWES

#### Tools
Προστέθηκε το μενού `Tools --> Security Files [BKP] Manager` από το οποίο μέσω του υπομενού του μπορείτε 
να πάρετε αντίγραφα σε zip των αρχείων ασφαλείας [.bkp] που δημιουργεί το AWES σε κάθε κρίσιμη αλλαγή 
που κάνετε μέσα από το παράθυρο διαμόρφωσης. 
Επίσης μπορείτε να διαγράψετε τα αρχεία αυτά από το AWES, αλλά και να επαναφέρετε ένα αντίγραφο zip, 
που θα επιλέξετε μέσα από ένα ειδικό πλαίσιο διαλόγου.

#### Logs
1) Προστέθηκε το μενού `Logs --> PHP Logs` με δυνατότητα προβολής και εκκαθάρισης των περιεχομένων των αρχείων καταγραφής 
όλων των ενεργών, στο `AWES`, εκδόσεων της PHP.

2) Προστέθηκε το μενού `Logs --> Databases Logs --> MariaDB Logs` με δυνατότητα προβολής και εκκαθάρισης των περιεχομένων 
των αρχείων καταγραφής όλων των ενεργών, στο AWES, εκδόσεων της βάσης δεδομένων MariaDB.

3) Προστέθηκε το μενού `Logs --> Databases Logs --> MongoDB Logs` με δυνατότητα προβολής και εκκαθάρισης των περιεχομένων 
των αρχείων καταγραφής όλων των ενεργών, στο AWES, εκδόσεων της βάσης δεδομένων MongoDB.  

#### Windows DNS Hosts
Προστέθηκε στο μενού `Servers --> DNS --> DNS Hosts File` επιλογή ανοίγματος στον Windows Explorer, 
του φακέλου που βρίσκεται το αρχείο hosts. 
Στο ίδιο μενού προστέθηκε επίσης, η δυνατότητα διαγραφής των αρχείων ασφαλείας που δημιουργούνται μετά από κάθε αλλαγή.

#### Configuration
1) Προστέθηκαν οι επιλογές διαγραφής ή καθαρισμού προσωρινών αρχείων και αρχείων καταγραφής κατά τον τερματισμό του AWES.

2) Διορθώθηκε στην καρτέλα `Users and Domains Settings --> Domains Settings --> Will domains have a password protection option?` η εντολή λειτουργίας ενεργοποίησης ή απενεργοποίησης στο πλαίσιο διαλόγου δημιουργίας ενός τομέα, της επιλογής `Do you want to confirm access to this domain?`. Τώρα όταν στο πλαίσιο διαμόρφωσης του AWES έχουμε επιλεγμένο το `Will domains have a password protection option?`, τότε θα μπορεί να λειτουργήσει και το `Do you want to confirm access to this domain?` , αλλιώς θα είναι ανενεργό και αυτό.

3) Διορθώθηκε στην καρτέλα `DNS Settings --> Windows DNS Hosts --> Enable Windows Hosts` 
η ενεργοποίηση και απενεργοποίηση των υπόλοιπων επιλογών στην καρτέλα, αναλόγως εάν είναι επιλεγμένη ή όχι η επιλογή αυτή.
Εφόσον γίνει αποθήκευση των αλλαγών στο Configuration, αυτόματα δημιουργεί και την ενεργοποίηση - απενεργοποίηση 
της αντίστοιχης καρτέλας και του κουμπιού `DNS` στην μπάρα ενεργειών.

4) Η Επιλογή `Users and Domains Settings --> Users Settings --> Force create a primary domain for a new user` 
μετακινήθηκε στην καρτέλα `Users and Domains Settings --> Domain Settings`.
Η επιλογή αυτή θα είναι πάντα τσεκαρισμένη ώστε σε κάθε νέο χρήστη να δημιουργούμε και τον κύριο τομέα του.


## ΔΙΑΚΟΜΙΣΤΕΣ

#### Filezilla
Έγινε αναβάθμιση του FTP Server στην έκδοση 1.9.4. 
Πρέπει να υπενθυμίσουμε πως η δωρεάν έκδοση που χρησιμοποιείται στο Ascoos Web Extended Studio, δεν υποστηρίζει SFTP.
Το αρχικό Password για είσοδο στην διαχείριση του Filezilla είναι το `root`.
Την διαμόρφωση την έχω αφήσει να γίνει απο τον κάθε χρήστη, διότι ο καθένας έχει διαφορετικές προσεγγίσεις. 


## ΠΡΟΣΘΕΤΑ

#### Mantis
Αναβαθμίστηκε στις εκδόσεις 2.25.8 για την PHP 5.6 και στην έκδοση 2.27.0 για PHP >= 7.4
Οι προηγούμενοι φάκελοι 20 και 26 δεν πειράζονται ώστε σε περίπτωση που έχετε κάνει κάποιες αλλαγές στο configuration του Mantis, να έχετε την δυνατότητα να το αντιγράψετε στους νέους φακέλους.
Τους παλιούς φακέλους 20 και 26 μπορείτε να τους διαγράψετε άφοβα (εφόσον έχετε πάρει backup και έχετε ανακτήσει τυχόν uploads).


## ΕΦΑΡΜΟΓΕΣ

#### Spintool
Έγινε προσθήκη της δωρεάν εφαρμογής spintool για να κάνετε τα απαραίτητα screenshots. Η εφαρμογή αυτή τοποθετήθηκε ως προεπιλογή και στο toolbar, ώστε να είναι άμεση η χρήση του.

#### Firefox Developer Edition
Αναβαθμίστηκε στην έκδοση 133.0b9


## ΜΗ ΔΙΟΡΘΩΜΕΝΑ

#### PHP 5.6.40 και 8.4
Λόγω της αναβάθμισης του apache και του openssl, δεν μπορεί να φορτώσει τους drivers του openssl και της MongoDB. (#0000001 - https://issues.ascoos.com/view.php?id=1).

#### SSL
Δεν ενεργοποιήθηκε η υποεφαρμογή παραγωγής πιστοποιητικών SSL για τα δημιουργούμενα domains, διότι παρουσιάζει ακόμη κάποια προβλήματα.
