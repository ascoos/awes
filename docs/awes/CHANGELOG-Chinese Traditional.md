# 24.2.6

⏰ 2025-10-25

## AWES
- 🛠️ `快捷工具列管理器`：已修正編輯快捷方式時的 `可執行路徑`、`參數` 和 `工作可執行路徑` 欄位。現在您可以修改這些欄位的內容。修改後請點擊 `儲存` 及 `套用`，以立即生效。

- ➕ `快捷方式`：新增多個快捷方式。
其中最值得注意的是執行 `chromium` 的快捷方式。如果您的 AWES 安裝在非 "C:\" 的磁碟機上，則在使用前需編輯該快捷方式，並在執行參數中更改磁碟機代號，以正確且可攜式地儲存 `預設設定檔`。
